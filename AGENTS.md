# AGENTS.md

This project's conventions for AI agents and contributors live in **[CLAUDE.md](./CLAUDE.md)**.

Read it before making changes. In short: follow Microsoft C# conventions, always use bracketed
control-flow blocks, one type per file, keep code simple, and develop test-first (TDD) with
behaviour-style specs (BDD) using xUnit + Shouldly + NSubstitute.
